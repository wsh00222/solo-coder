# 📚 个人书籍收藏管理系统

一个现代化的个人书籍收藏管理应用，使用 React + Express 构建。

## ✨ 功能特性

- **书籍管理**：添加、编辑、删除书籍，支持完整字段（书名、作者、出版年份、类型、评分、阅读状态）
- **卡片展示**：美观的卡片列表展示，含星星评分（只读）和悬停高亮效果
- **统计面板**：总藏书量、各状态数量、平均评分（保留一位小数）实时更新
- **筛选排序**：按状态/类型下拉筛选，按书名/评分升序/降序排序
- **详情页面**：点击卡片进入详情页，提供编辑和删除（二次确认）
- **数据校验**：评分必须为 1-5 整数，所有字段必填校验
- **分页功能**：每页 6 本，总数大于 6 时自动显示分页控件
- **空状态引导**：无书籍时显示友好的引导页面
- **示例数据**：首次启动自动生成 4 本示例书籍（覆盖不同状态和类型）

## 🛠️ 技术栈

- **前端**：React 18 + React Router 6 + Vite
- **后端**：Node.js + Express 4
- **数据持久化**：JSON 文件存储
- **样式**：原生 CSS + 现代渐变设计

## 🚀 快速开始

### 1. 安装依赖

```bash
# 安装后端依赖
cd server
npm install

# 安装前端依赖（新开终端）
cd ../client
npm install
```

### 2. 启动服务

**启动后端服务器**（端口 3001）：
```bash
cd server
npm start
```

**启动前端开发服务器**（端口 5173）：
```bash
cd client
npm run dev
```

### 3. 访问应用

打开浏览器访问：http://localhost:5173

## 📖 API 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/books/init` | 初始化示例数据 |
| GET | `/api/books` | 获取所有书籍 |
| GET | `/api/books/:id` | 获取单本书籍详情 |
| POST | `/api/books` | 新增书籍 |
| PUT | `/api/books/:id` | 更新书籍信息 |
| DELETE | `/api/books/:id` | 删除书籍 |

### 书籍数据模型

```json
{
  "id": "uuid",
  "title": "书名",
  "author": "作者",
  "year": 2024,
  "genre": "小说",
  "rating": 5,
  "status": "读完"
}
```

**字段说明：**
- `genre`（类型）：小说 / 科技 / 生活 / 其他
- `rating`（评分）：1-5 整数
- `status`（状态）：想读 / 在读 / 读完
